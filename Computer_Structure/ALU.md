# Arithmetic Logic Unit
## 산술 논리 장치

산술 연산과 논리 연산을 수행하는 장치이다.

학부 2학년때, 문득 Big-O표기법을 보면서<br/>
같은 결과를 도출하는 덧셈 여러번과 곱셈 한번을 요청할 때 어느 것이 빠를까 생각해본 적 있는데<br/>
생각보다 큰 차이는 나지 않았고

나중에 컴퓨터 구조를 배우면서 어차피 IO작업에서 상당한 시간이 소요되기에 큰 차이가 나지 않는다는 것을 깨달았다.<br/>
